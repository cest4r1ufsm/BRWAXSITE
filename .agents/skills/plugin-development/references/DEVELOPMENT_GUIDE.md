# Plugin Development Guide

## Workflow
1. **Design**: Plan agents, skills, commands
2. **Implement**: Create YAML frontmatter + content
3. **Validate**: Run plugin validator
4. **Test**: Verify in Claude Code
5. **Publish**: Add to marketplace

## Best Practices
- Follow SASMP v1.3.0
- Use Golden Format for skills
- Keep agents focused (single responsibility)
